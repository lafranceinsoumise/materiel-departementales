Bulletin

Pour générer le Bulletin de Vote :

  • Entrez les noms des candidat·es tels qu'ils seront déposés en préfecture. Inscrivez les noms de telle sorte que les candidates soient dans la colonne de gauche du bulletin, et les candidats dans la colonne de droite.
  • Cliquez sur "mettre à jour" pour vérifier les informations, puis sur "télécharger". Vous obtiendrez un fichier PNG qui sera téléchargé dans le dossier "Téléchargements" de votre ordinateur.
  • Allez sur le site https://png2pdf.com/
  • Ouvrez le dossier Téléchargements de votre ordinateur, trouvez le fichier PNG du bulletin de vote que vous venez de générer et glissez-déposez-le sur le rectangle vert pâle.
  • Attendez la fin de la conversion.
  • Cliquez sur le rectangle jaune "DOWNLOAD". Vous obtiendrez un fichier PDF qui sera téléchargé dans le dossier "Téléchargements" de votre ordinateur. Vous pouvez transmettre ce fichier à votre imprimeur.

Lancer le générateur de bulletin

Professions de foi

Pour générer la profession de foi

  • Entrez le nom du département et du canton
  • Entrez le nom du premier candidat tel qu'il sera déposé en préfecture. Entrer la profession du candidat dans le troisième champ.
  • Cliquez sur "Parcourir" et sélectionnez dans votre ordinateur la photo correspondant. La photo doit être en format CARRÉ.
  • Réitérez l'opération pour tous les autres candidats
  • Cliquez sur "prévisualiser" pour vérifier les informations, puis sur "télécharger". Vous obtiendrez un fichier PNG qui sera téléchargé dans le dossier "Téléchargements" de votre ordinateur.
  • Cliquez sur le lien suivant pour obtenir le verso de la profession de foi : https://cloud.franceinsoumise.org/index.php/s/jYzjC8238LJwS4J.
  • Cliquez sur "Télécharger" en haut à droite. Vous obtiendrez un fichier PNG qui sera téléchargé dans le dossier "Téléchargements" de votre ordinateur.
  • Allez sur le site https://png2pdf.com/
  • Glissez-déposez les DEUX fichiers PNG que vous venez de générer sur le rectangle vert et attendez la fin de la conversion.
  • Cliquez sur le rectangle jaune "DOWNLOAD". Vous obtiendrez un fichier PDF qui sera téléchargé dans le dossier "Téléchargements" de votre ordinateur. Vous pouvez transmettre ce fichier à votre imprimeur.

Lancer le générateur de professions de foi
